# Thawed [![NuGet](https://img.shields.io/nuget/v/Thawed.svg)](https://www.nuget.org/packages/Thawed/) [![Downloads](https://img.shields.io/nuget/dt/Thawed)](https://www.nuget.org/packages/Thawed/) [![GitHub builds](https://github.com/klaumit/Thawed/actions/workflows/dotnet.yml/badge.svg)](https://github.com/klaumit/Thawed/actions)

<img align="right" width="160px" height="160px" src="logo.png">

This is an Intel 80186 (or NEC compatible) instruction decoder and disassembler.

## Overview
This project should disassemble the following:

| Vendor     | Example chips           |
| -----------|-------------------------|
| Intel      | 80C186, 80186, 8086     |
| AMD        | A80186, Am186, Am8086   |
| NEC        | V20, V30(MZ), V40, V50  |
| Siemens    | SAB80186, SAB8086       |
| Fujitsu    | MBL80186                |
| Rochester  | MG80186, MG80C186XL     |
| CAST       | C80186TX                |
| Jamie Iles | S80186                  |
